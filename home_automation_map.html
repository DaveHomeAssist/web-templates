Homelab notes

Tailscale + HA: Expose your instance remotely without port forwarding. Combine with HA Cloud for voice assistant fallback when off-VPN.

Recommended stack: HA on Dell server (Proxmox VM or dedicated SSD), Zigbee2MQTT on same host, MQTT broker local, Dasherr on separate container for dashboard aggregation.
